How do I get credentials?

Server admin sets these via env vars:

STACKLY_API_KEYS=sk_live_abc,sk_live_xyz:pro
STACKLY_BASIC_USERS=alice:wonderland
STACKLY_JWT_SECRET=your-256-bit-secret

Install Chrome Extension

Manifest V3 · 5 steps

  1. 1
    Download the .zip
    Click the green button below to save stackly-chrome-extension.zip
  2. 2
    Unzip to a permanent folder
    e.g. ~/tools/stackly/ — don't move it after install
  3. 3
    Open chrome://extensions
    Toggle Developer mode in the top-right corner
  4. 4
    Click "Load unpacked"
    Select the unzipped folder (the one containing manifest.json)
  5. 5
    Pin and use
    Click the puzzle icon → pin Stackly → click the S badge on any site
